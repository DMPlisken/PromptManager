Install the Agent

Run these commands on the target machine (macOS Terminal).
Step 1: Install the agent
npm install -g @promptflow/agentCopy
Step 2: Pair with this server
promptflow-agent pair A3X9K2 --server ws://192.168.1.50:3001Copy
✦ Easiest: Paste this into Claude Code CLI
Open Claude Code on the target machine and paste this prompt. It will install, configure, pair, and start the agent automatically.
Install and configure the PromptFlow remote agent on this machine. Do everything automatically without asking questions. Steps: 1. Clone the agent: git clone ... 2. npm install 3. Create config: ~/.promptflow/agent.yaml 4. Register with server 5. Start the agentCopy Prompt
Tip: The Claude Code prompt above handles everything — install, config, pairing, and verification. Just paste it and let Claude do the work.
