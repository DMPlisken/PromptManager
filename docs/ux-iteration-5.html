UX Iteration 5 — History Page & Typography

Focus: Execution history card redesign, typography consistency, global spacing refinements

Theme

The first half of the iteration series (1-4) addressed the primary pages. Iteration 5 turned attention to the History page and cross-cutting typography and spacing consistency that affects every page.

Changes Made

History Page Polish

  • Execution records now use card-based layout matching the session sidebar style
  • Group name displayed as a colored badge matching the group's theme
  • Variable values shown as pill-shaped tags with name:value format
  • Expanded view uses a monospace font for the rendered prompt text
  • Date formatting standardized to relative time ("2h ago") with full timestamp on hover

Typography Consistency

  • Standardized heading sizes: H1 (28px), H2 (20px), H3 (16px) across all pages
  • Body text consistently 14px with 1.6 line-height
  • Monospace font stack: Menlo, Consolas, Courier New for code/terminal content
  • Muted text color (--text-dim) used consistently for secondary information

Global Spacing Refinements

  • Card padding standardized to 20px on all components
  • Section gaps standardized to 16px between cards, 32px between sections
  • Button padding normalized to 8px 16px for regular, 6px 12px for compact
  • Form field spacing set to 12px gap between fields
